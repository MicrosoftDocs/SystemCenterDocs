# Step 6: Extend the Change Area Enumeration List
In this step of the Woodgrove Bank customization scenario, Ken extends the **Change Area Enumeration** list by adding a new **Compliance** list item that represents the new change request type.  
  
 The following procedure provides only the high\-level steps for creating a new list item in the [!INCLUDE[smcons](../../../sm/deploy/deploy-guide/includes/smcons_md.md)]. For the complete procedure for creating a new list item, see [How to Add a List Item](http://go.microsoft.com/fwlink/p/?LinkId=231881).  
  
### To create a new list item  
  
1.  In the [!INCLUDE[smcons](../../../sm/deploy/deploy-guide/includes/smcons_md.md)], select **Change Area Enumeration** as the list to edit.  
  
2.  Specify **Compliance** as the new value to add to this list.
