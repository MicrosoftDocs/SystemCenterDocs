# Personalizing the Activities Toolbox
Use the procedures in this section to personalize the **Activities Toolbox** pane in the [!INCLUDE[smatfull2012](../../../sm/manage/author/includes/smatfull2012_md.md)]. Each user can personalize the **Activities Toolbox** differently; the [!INCLUDE[scauthoringshort](../../../sm/manage/author/includes/scauthoringshort_md.md)] stores this information as part of each userâ€™s profile.  
  
> [!IMPORTANT]  
>  You can only modify personalized activity groups. You cannot modify the default activity groups.
