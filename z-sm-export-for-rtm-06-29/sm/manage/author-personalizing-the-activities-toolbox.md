# Personalizing the Activities Toolbox
Use the procedures in this section to personalize the **Activities Toolbox** pane in the System Center 2016 - Service Manager Authoring Tool. Each user can personalize the **Activities Toolbox** differently; the Authoring Tool stores this information as part of each userâ€™s profile.  
  
> [!IMPORTANT]  
>  You can only modify personalized activity groups. You cannot modify the default activity groups.
