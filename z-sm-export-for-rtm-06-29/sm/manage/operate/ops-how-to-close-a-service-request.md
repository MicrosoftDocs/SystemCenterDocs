# How to Close a Service Request
In [!INCLUDE[smlong12](../../../sm/deploy/deploy-guide/includes/smlong12_md.md)], after all the review activities are approved or rejected and any manual activities are completed, you can close a service request in the [!INCLUDE[smcons](../../../sm/deploy/deploy-guide/includes/smcons_md.md)]. You can use the following procedure to close a service request.  
  
### To close a service request  
  
1.  In the [!INCLUDE[smcons](../../../sm/deploy/deploy-guide/includes/smcons_md.md)], click **Work Items**.  
  
2.  In the **Work Items** pane, expand **Work Items**, expand **Service Request Fulfillment**, and then click **Completed Service Requests**.  
  
3.  Select a completed service request. For example, select **Request Membership to Active Directory Group â€“ Joe Andreshak**.  
  
4.  In the **Tasks** pane, click **Close**.  
  
5.  In the **Comments** dialog box, type any comments that you have for the closure, and then click **OK**.
