# How to Remove an Activity from a Workflow
Use this procedure to remove an activity from a workflow in the System Center 2016 - Service Manager Authoring Tool. This operation does not remove the activity from the Activity Library or from the **Activities Toolbox** pane.  
  
### To remove an activity from a workflow  
  
-   In the authoring pane, right\-click the activity, and then click **Delete**.
