  <introduction>
    <para>IPAM in <token>win8_server_./Token> and <token>winblue_server_./Token> is a new built-in framework for discovering, monitoring, auditing, and managing the IP address space used on a corporate network. IPAM provides for administration and monitoring of servers running Dynamic Host Configuration Protocol (DHCP) and Domain Name System (DNS).</para>
