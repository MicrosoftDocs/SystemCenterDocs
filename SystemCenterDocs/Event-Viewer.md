# Event Viewer
<?xml version="1.0" encoding="utf-8"?>
<developerConceptualDocument xmlns="http://ddue.schemas.microsoft.com/authoring/2003/5" xmlns:xlink="http://www.w3.org/1999/xlink" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://ddue.schemas.microsoft.com/authoring/2003/5 http://dduestorage.blob.core.windows.net/ddueschema/developer.xsd">
  <introduction>
    <para>The Event Viewer is a Microsoft Management Console (MMC) snap-in that enables you to browse and manage event logs. It is an indispensable tool for monitoring the health of systems and troubleshooting issues when they arise. </para>
    <para>Event Viewer enables you to perform the following tasks:</para>
    <list class="bullet">
      <listItem>
        <para>View events from multiple event logs</para>
      </listItem>
      <listItem>
        <para>Save useful event filters as custom views that can be reused</para>
      </listItem>
      <listItem>
        <para>Schedule a task to run in response to an event</para>
      </listItem>
      <listItem>
        <para>Create and manage event subscriptions</para>
      </listItem>
    </list>
    <para>For detailed information about how to use the Event Viewer, see <externalLink><linkText>Event Viewer How To...</linkText><linkUri>http://technet.microsoft.com/library/cc749408.aspx</linkUri></externalLink></para>
  </introduction>
