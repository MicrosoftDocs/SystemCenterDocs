# Day-to-day tasks in System Center 2016 - Service Manager

>Applies To: System Center 2016 - Service Manager

The articles in this section provide information about using the Service Manager console for day\-to\-day tasks that you perform to manage service desk functions for your organization's computer infrastructure. There articles provide detailed procedures for the following tasks:  

- Search for information about configuration items and other incidents, problems, and knowledge articles in Service Manager.
- Manage incidents and problems in Service Manager by creating, editing, and resolving them.
- Manage changes and activities in Service Manager by creating, editing, and closing them.  
- Manage service requests in Service Manager by creating, approving, fulfilling, and closing them.  
- Manage release records in Service Manager by creating, editing, combining, and using them to track releases.  
- Use data warehouse reporting and analytics in Service Manager to view data and trends across your Service Manager environment.
