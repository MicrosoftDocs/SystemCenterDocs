# Service Manager Connectors

>Applies To: System Center 2016 - Service Manager

Any connectors that you created with System Center 2012 R2 - Service Manager will continue to function after you upgrade to System Center 2016 - Service Manager. For more information, see [Using Connectors to Import Data into Service Manager](../manage/admin-using-connectors-to-import-data-into-system-center-2016-service-manager).
