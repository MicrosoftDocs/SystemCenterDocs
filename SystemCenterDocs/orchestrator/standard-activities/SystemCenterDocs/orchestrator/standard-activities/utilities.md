# Utilities
The following table provides a brief description of tasks you can accomplish when using each Utilities activity.  
  
|Tasks|Utilities Activities|  
|-----------|--------------------------|  
|Transform XML files.|[Apply XSLT](../../../SystemCenterDocs/orchestrator/standard-activities/apply-xslt.md)|  
|Perform an XPath query on an XML file.|[Query XML](../../../SystemCenterDocs/orchestrator/standard-activities/query-xml.md)|  
|Transform existing published data or variable items into new content.|[Map Published Data](../../../SystemCenterDocs/orchestrator/standard-activities/map-published-data.md)|  
|Compare two text values or two numerical values and determine whether or not they are equal.|[Compare Values](../../../SystemCenterDocs/orchestrator/standard-activities/compare-values.md)|  
|Create or add information to an HTML file.|[Write Web Page](../../../SystemCenterDocs/orchestrator/standard-activities/write-web-page.md)|  
|Read lines in a structured text log file.|[Read Text Log](../../../SystemCenterDocs/orchestrator/standard-activities/read-text-log.md)|  
|Write a row into a database table.|[Write to Database](../../../SystemCenterDocs/orchestrator/standard-activities/write-to-database.md)|  
|Query a database and return the resulting rows as published data.|[Query Database](../../../SystemCenterDocs/orchestrator/standard-activities/query-database.md)|  
|Invoke a runbook when a counter has reached a specific value.|[Monitor Counter](../../../SystemCenterDocs/orchestrator/standard-activities/monitor-counter.md)|  
|Retrieve the value of a counter and return it as a published data item.|[Get Counter Value](../../../SystemCenterDocs/orchestrator/standard-activities/get-counter-value.md)|  
|Increment and decrement a counter and reset it to its default value.|[Modify Counter](../../../SystemCenterDocs/orchestrator/standard-activities/modify-counter.md)|  
|Run a web service with XML parameters you specify.|[Invoke Web Services](../../../SystemCenterDocs/orchestrator/standard-activities/invoke-web-services.md)|  
|Transform existing date and time formats into customized formats.|[Format Date/Time](../../../SystemCenterDocs/orchestrator/standard-activities/format-date-time.md)|  
|Generate random strings of text.|[Generate Random Text](../../../SystemCenterDocs/orchestrator/standard-activities/generate-random-text.md)|  
|Map a network path.|[Map Network Path](../../../SystemCenterDocs/orchestrator/standard-activities/map-network-path.md)|  
|Disconnect a network path.|[Disconnect Network Path](../../../SystemCenterDocs/orchestrator/standard-activities/disconnect-network-path.md)|  
|Connect to a dial-up or VPN network.|[Connect/Disconnect Dial-up](../../../SystemCenterDocs/orchestrator/standard-activities/connect-disconnect-dial-up.md)|  
|Get the status of a dial-up or VPN network.|[Get Dial-up Status](../../../SystemCenterDocs/orchestrator/standard-activities/get-dial-up-status.md)|
