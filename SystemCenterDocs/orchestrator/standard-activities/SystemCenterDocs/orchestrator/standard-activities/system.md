# System
The following table provides a brief description of tasks you can accomplish when using each System activity.  
  
|Tasks|System Activities|  
|-----------|-----------------------|  
|Run any program or command on any computer in your domain.|[Run Program](../../../SystemCenterDocs/orchestrator/standard-activities/run-program.md)|  
|Run scripts that parse data or run functions against available APIs.|[Run .Net Script](../../../SystemCenterDocs/orchestrator/standard-activities/run-net-script.md)|  
|End processes that are running on the runbook server or on a remote computer.|[End Process](../../../SystemCenterDocs/orchestrator/standard-activities/end-process.md)|  
|Start, stop, pause, or restart a Windows service.|[Start/Stop Service](../../../SystemCenterDocs/orchestrator/standard-activities/start-stop-service.md)|  
|Restart a computer on your network.|[Restart System](../../../SystemCenterDocs/orchestrator/standard-activities/restart-system.md)|  
|Save entries from an event log so that they can be used later.|[Save Event Log](../../../SystemCenterDocs/orchestrator/standard-activities/save-event-log.md)|  
|Send a Windows Management Instrumentation (WMI) query to a system that you specify and then return the results.|[Query WMI](../../../SystemCenterDocs/orchestrator/standard-activities/query-wmi.md)|  
|Open an SSH connection to a remote server and run shell commands on that server.|[Run SSH Command](../../../SystemCenterDocs/orchestrator/standard-activities/run-ssh-command.md)|  
|Query a network device for the value of variable that is assigned to the Management Information Base (MIB) address you specify.|[Get SNMP Variable](../../../SystemCenterDocs/orchestrator/standard-activities/get-snmp-variable.md)|  
|Wait for an event to occur either in the Microsoft SNMP Trap Service or on a port that you specify.|[Monitor SNMP Trap](../../../SystemCenterDocs/orchestrator/standard-activities/monitor-snmp-trap.md)|  
|Raise an SNMP event that can be detected by a network systems manager application.|[Send SNMP Trap](../../../SystemCenterDocs/orchestrator/standard-activities/send-snmp-trap.md)|  
|Modify a variable that is specified by its Management Information Base (MIB).|[Set SNMP Variable](../../../SystemCenterDocs/orchestrator/standard-activities/set-snmp-variable.md)|
