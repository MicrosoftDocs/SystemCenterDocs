# Scale Tier In activity

::: moniker range=">= sc-orch-1801 <= sc-orch-1807"

[!INCLUDE [eos-notes-orchestrator.md](../includes/eos-notes-orchestrator.md)]

::: moniker-end

The Scale Tier In activity is used in a runbook to remove one virtual machine instance from a specified service tier.

The activity publishes all the data from the required and optional properties into published data. The following tables list the required and optional properties and published data for this activity.

## Required properties

**Element**<br>Service Name<br>Tier Name

## Optional properties

This activity has no optional properties.

## Published data

This activity has no published data.
