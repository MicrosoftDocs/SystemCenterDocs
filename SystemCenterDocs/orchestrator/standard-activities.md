# Standard activities

Standard Activities enable you to create runbooks to automate your data center procedures.
Activities are organized into categories to help you find the appropriate activity for the task you want to perform. [Learn more](https://msdn.microsoft.com/en-us/library/hh225055).


[Here](https://msdn.microsoft.com/en-us/library/hh225046.aspx) is the detailed list of Orchestrator standard activities.

The following table provides a brief description of tasks you can accomplish with each activity category.  
  
|Tasks|Categories|  
|-----------|----------------|  
|Run system commands.|[System](../../../SystemCenterDocs/orchestrator/standard-activities/system.md)|  
|Perform schedule-based activities.|[Scheduling](../../../SystemCenterDocs/orchestrator/standard-activities/scheduling.md)|  
|Monitor processes or system-level events.|[Monitoring](../../../SystemCenterDocs/orchestrator/standard-activities/monitoring.md)|  
|Manage file interactions such as copying and moving files.|[File Management](../../../SystemCenterDocs/orchestrator/standard-activities/file-management.md)|  
|Send e-mail notifications.|[Email](../../../SystemCenterDocs/orchestrator/standard-activities/email.md)|  
|Support other notification types.|[Notification](../../../SystemCenterDocs/orchestrator/standard-activities/notification.md)|  
|Search for or modify data within a workflow.|[Utilities](../../../SystemCenterDocs/orchestrator/standard-activities/utilities.md)|  
|Manipulate text files.|[Text File Management](../../../SystemCenterDocs/orchestrator/standard-activities/text-file-management.md)|  
|Manage workflows.|[Runbook Control](../../../SystemCenterDocs/orchestrator/standard-activities/runbook-control.md)|
