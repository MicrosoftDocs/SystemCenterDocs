## New features in Orchestrator 2025

The following sections introduce new and updated features in System Center Orchestrator 2025.

### Support for OLEDB19

SCO 2025 supports OLEDB19  which encrypts SQL Connection by default. Ensure to have Trusted Server certificate for successful SQL connection. [Learn more](/system-center/orchestrator/install?view=sc-orch-2025#secure-connection-to-sql-server&preserve-view=true).

### Support for .NET 8

System Center Orchestrator 2025 supports the latest and most secure .NET8 (LTSC). [Learn more](/system-center/orchestrator/system-requirements-orch?view=sc-orch-2025&preserve-view=true).

### Support for Transport Security Layer (TLS) protocol version 1.3

System Center Orchestrator 2025 supports Transport Security Layer (TLS) protocol version 1.3.

### Support for Group Managed Service Accounts (gMSA)

System Center Orchestrator now supports Group Managed Service Accounts (gMSA). [Learn more](/windows-server/identity/ad-ds/manage/group-managed-service-accounts/group-managed-service-accounts/group-managed-service-accounts-overview).

## New features in Orchestrator 2025 UR1

The following sections introduce the new features and feature updates supported in Orchestrator 2025 Update Rollup 1 (UR1).

### Support for SQL Server 2025

System Center Orchestrator 2025 UR1 supports SQL Server 2025. [Learn more](/system-center/orchestrator/system-requirements-orch?view=sc-orch-2025#sql-server).
