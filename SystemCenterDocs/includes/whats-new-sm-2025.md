## New features in SM 2025

See the following sections for information about the new features/features updated in Service Manager (SM) 2025. It also includes the new features in Service Manager 2025 UR1.

### Support for Windows Server 2025  

Service Manager 2025 now supports Windows Server 2025. [Learn more](/system-center/scsm/system-requirements?view=sc-sm-2025&preserve-view=true).

### Other updates

Service Manager 2025 also includes the following updates:

Supports the following newer browsers Chrome and Edge:

- Microsoft Edge version 121 and later with IE compatibility mode.
- Google Chrome version 121 and later.

## New features in Service Manager 2025 UR1

The following sections introduce the new features or feature updates supported in Service Manager 2025 Update Rollup 1 (UR1).

For problems fixed in UR1 and the installation instructions for UR1, see the [KB article](https://support.microsoft.com/kb/5068305).

### Support for SQL 2025 

Service Manager 2025 UR1 supports SQL 2025. [Learn more](/system-center/scsm/sm-sql-reqs?view=sc-sm-2025).

### Support for MSOLEDBSQL 19 

Service Manager 2025 UR1 supports MSOLEDBSQL 19. [Learn more](/system-center/scsm/sm-sql-reqs?view=sc-sm-2025).

### Support for SCO Web API 

Service Manager 2025 UR1 supports connecting to System Center Orchestrator Web API 2022 or 2025, while retaining compatibility with the existing SCO 2019 Web Service. Learn more.
