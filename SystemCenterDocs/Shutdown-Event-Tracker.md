# Shutdown Event Tracker
<?xml version="1.0" encoding="utf-8"?>
<developerConceptualDocument xmlns="http://ddue.schemas.microsoft.com/authoring/2003/5" xmlns:xlink="http://www.w3.org/1999/xlink" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://ddue.schemas.microsoft.com/authoring/2003/5 http://dduestorage.blob.core.windows.net/ddueschema/developer.xsd">
  <introduction>
    <para>Shutdown Event Tracker records the reason why you shut down or restart a computer so that system administrators can review the data later. System administrators can require that you provide this information when you shut down or restart the computer, or that you provide information when you are the first user to sign in to the computer after an unexpected restart or shutdown. </para>
    <para>For detailed information on how to use Shutdown Event Tracker, see <externalLink><linkText>Work with Shutdown Event Tracker</linkText><linkUri>http://technet.microsoft.com/library/cc772534.aspx</linkUri></externalLink>.</para>
    <para>You can also provide a reason why you shut down or restart a computer when you use the <system>shutdown</system> command. For more information, see <externalLink><linkText>Shutdown</linkText><linkUri>http://go.microsoft.com/fwlink/?LinkID=244423</linkUri></externalLink> in the Windows Server Technical Library.</para>
  </introduction>
