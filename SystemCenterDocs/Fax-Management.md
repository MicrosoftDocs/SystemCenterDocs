    <para>Fax Service Manager provides a central administration point for configuring and managing your fax resources. Fax Service Manager is installed when you install the Fax Server role in Server Manager. You can use Fax Service Manager to configure fax devices for incoming and outgoing fax traffic, specify who can use a fax device, set routing rules for incoming and outgoing faxes, configure a fax archiving policy, manage the Fax service, specify security settings, and log and monitor fax activity.</para>
