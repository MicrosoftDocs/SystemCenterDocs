# Using Connectors to Import Data into System Center 2012 - Service Manager
You can use [!INCLUDE[smlong12](./Token/smlong12_md.md)] connectors to import data as configuration items from Active Directory Domain Services \(AD DS\), System Center Configuration Manager, [!INCLUDE[orchlong](./Token/orchlong_md.md)], [!INCLUDE[vmm12long](./Token/vmm12long_md.md)], and System Center Operations Manager. In addition, you can import alerts from Operations Manager, and you can configure these alerts to automatically generate incidents in [!INCLUDE[smshort](./Token/smshort_md.md)]. You can also import data from comma\-separated value \(CSV\) files into the [!INCLUDE[smshort](./Token/smshort_md.md)] database.
