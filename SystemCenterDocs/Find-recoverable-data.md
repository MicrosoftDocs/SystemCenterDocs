# Find recoverable data
You can easily and quickly recover data from the recovery points stored on the server for [!INCLUDE[dpm2012long](./Token/dpm2012long_md.md)]. To find your data, browse through the recovery points of protected data sources or search to locate the specific files that you want to recover. After you find the data that you want to recover, you can choose the version that you want to recover.
