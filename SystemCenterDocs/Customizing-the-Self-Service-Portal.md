# Customizing the Self Service Portal
You can customize the [!INCLUDE[smssp](Token/smssp_md.md)] in [!INCLUDE[smlong12](Token/smlong12_md.md)] to better suit your organization. Example customizations presented in this section are somewhat limited; they include changes to formatting and adding a SharePoint list. However, the [!INCLUDE[smssp](Token/smssp_md.md)] is based on SharePoint web services, so you can make additional customizations on your own, outside the scope of this section.

Use the procedures in the following topics to customize the [!INCLUDE[smssp](Token/smssp_md.md)].

## [!INCLUDE[smssp](Token/smssp_md.md)] Customization Topics

-   [How to Update the Portal Title and Image](How-to-Update-the-Portal-Title-and-Image.md)

    Describes how to update the [!INCLUDE[smssp](Token/smssp_md.md)]’s tile and associated logo.

-   [How to Add Navigation Links](How-to-Add-Navigation-Links.md)

    Describes how to add navigational links to the [!INCLUDE[smssp](Token/smssp_md.md)].

-   [How to Update the Portal Theme](How-to-Update-the-Portal-Theme.md)

    Describes how to update the theme for the entire [!INCLUDE[smssp](Token/smssp_md.md)].

-   [How to Add a Portal Announcement Page](How-to-Add-a-Portal-Announcement-Page.md)

    Describes how to add an announcement page using a SharePoint list.

-   [How to Modify the Help Article Web Part](How-to-Modify-the-Help-Article-Web-Part.md)

    Describes how to modify a specific web part in the [!INCLUDE[smssp](Token/smssp_md.md)].

-   [How to Modify a Master Page File](How-to-Modify-a-Master-Page-File.md)

    Describes how to modify a master page file using SharePoint Designer.

-   [How to Select the Self Service Portal Language](How-to-Select-the-Self-Service-Portal-Language.md)

    Describes how to select the localized languages that are displayed in the [!INCLUDE[smssp](Token/smssp_md.md)] based on a user’s choice.
